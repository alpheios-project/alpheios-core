Alpheios WebExtension Demo Page

This pages serves as a demonstrator of Alpheios WebExtension. First, you need to load a WebExtensions as a temporary extension using browser's extensions UI. This will load an extension's background script into a browser. Second, you need to load a content part of an extension into an active page. For that, right click on a page and select "Load Alpheios" form the menu that will appear. This will load an Alpheios panel (hidden by default) and Alpheios page controls (clicking on them will either open or hide a panel). To load word data, double click on a word from a text below. Only words from test cases will return any data at the moment. They are highlighted with yellow. All other words will return a message saying that data of this word is not found.
